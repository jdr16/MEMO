# 数制间的相互转换

*二进制和十进制 

| 十进制 |   二进制    | 十六进制 |
| :-: | :------: | ---- |
| 94  | 01011101 | 95H  |

*在程序设计中，经常会遇到需要将**二进制数转化成十进制数***
- 因为CPU只识别二进制数，但是人类更容易理解十进制数
- 因此需要将CPU的输出从二进制转化成为十进制数

**将二进制数转化为十进制数的方法：**
- 除余法
	- 0x93 --> 用它除0xA，从而求得十进制表示
- 比较法

**输入的十进制数转化为二进制数的方法**
- 输入的二进制数，比如敲进去了 95 
- 实际上计算机的运算是： (0x9H)\*(0xAH)+(0x5H)
- 所以其实我们敲进去的数，直接是用二进制表示的，输入多位数据的时候，计算机只是做了一个简单的数学运算
